# Mdit

Mdit 把 Notion 式的编辑体验和 Obsidian 式的文件掌控感放到了一起。

它带来流畅、结构化的写作体验，同时让你的笔记继续以本地 Markdown 文件的形式留在磁盘里。没有锁死的数据库，不强制上云，也不用在现代编辑器体验和自己能长期掌控的文件之间做取舍。

Mdit 的核心方向很明确：

- 始终保留在你设备上的纯 `.md` 文件
- 适合思考和整理内容的 Notion 风格编辑体验
- 在类似 Obsidian 的本地 Markdown 仓库中使用反向链接和图谱视图
- 以隐私优先，可用你自己的 API 密钥或本地模型运行的 AI

---

## Why Mdit

| 特性 | Mdit | Notion | Obsidian |
| --- | --- | --- | --- |
| 本地文件 | 是 | 否 | 是 |
| 块编辑器 | 是 | 是 | 否 |
| 反向链接和图谱视图 | 是 | 否 | 是 |
| 同步 | 计划中 | 是 | 是 |
| AI | 使用你自己的 API 密钥或本地模型 | 云优先 | 依赖插件 |
| 开源 | 是 | 否 | 否 |

Mdit 适合这样的人：喜欢现代工作空间工具的写作体验，但也希望文件始终握在自己手里。
