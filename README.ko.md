# Mdit

Mdit는 노션 같은 UX와 옵시디언 같은 파일 소유권을 함께 가져가는 앱입니다.

매끄럽고 구조화된 글쓰기 경험을 제공하면서도, 노트는 내 디스크에 있는 로컬 Markdown 파일로 그대로 남습니다. 잠긴 데이터베이스도 없고, 클라우드를 강요하지도 않습니다. 현대적인 에디터 경험과 내가 직접 관리하는 파일 사이에서 하나를 포기할 필요가 없습니다.

Mdit는 이런 방향을 중심에 두고 있습니다:

- 내 기기에 그대로 남는 `.md` 파일
- 생각을 정리하고 구조를 잡기 좋은 노션 스타일 에디팅 경험
- 옵시디언 같은 로컬 Markdown 볼트에서 쓰는 백링크와 그래프 뷰
- 내 API 키나 로컬 모델로 직접 설정해서 쓸 수 있는 프라이버시 중심 AI

## Why Mdit

| 기능 | Mdit | Notion | Obsidian |
| --- | --- | --- | --- |
| 로컬 파일 | 예 | 아니오 | 예 |
| 블록 에디터 | 예 | 예 | 아니오 |
| 백링크와 그래프 뷰 | 예 | 아니오 | 예 |
| 동기화 | 예정 | 예 | 예 |
| AI | 내 API 키 또는 로컬 모델 | 클라우드 중심 | 플러그인 의존 |
| 오픈소스 | 예 | 아니오 | 아니오 |

Mdit는 모던한 글쓰기 경험은 원하지만, 파일 소유권까지 넘기고 싶지는 않은 사람을 위한 도구입니다.
